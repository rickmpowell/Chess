6

8 - previous method
9 - next method

bs

Y - REDO
U - word left
I - line up
O - word right
P - start of file
[ - goto declaration
] - goto definition
\

H - start of line (toggles between start of line and first indent)
J - character left
K - line down
L - character right
; - end of line 
' - goto 
    <ctrl-'>F - goto file
    <ctrl-'>T - goto type
cr

N - matching brace
M
,  
.
/ - end of file
